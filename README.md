- 기존 서비스의 단점 -> 본 프로젝트의 장점
- 본 웹서비스의 사용자가 경험할 효과 및 장점

##### 4. 프로젝트 구성

- 사용 스택
  - 프론트엔드
    - React
    - 이외
  - 백엔드
    - Express, Node
    - 이외
- 와이어프레임 (Figma 등 링크 삽입)
- 스토리보드 및 유저 시나리오
  - 홈 화면에서 어떤 버튼을 클릭하여 어떤 페이지로 이동할 수 있다.
  - 이 버튼을 클릭하면 이런 웹 서비스가 실행된다.
